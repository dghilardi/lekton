# Security Policy

## Supported Versions

Currently, Lekton is in early development (alpha). Security updates will be provided on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of Lekton seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1.  **Do not open a public issue.**
2.  Send an email to lekton@d71.dev with details of the vulnerability.
3.  Include a brief description, steps to reproduce, and potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix. We ask you to follow responsible disclosure practices and give us time to address the issue before making it public.
