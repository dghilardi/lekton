# Getting Started with Lekton

Welcome to Lekton! This is a demo document to help you get started.

## What is Lekton?

Lekton is a high-performance, dynamic Internal Developer Portal (IDP). It is designed to replace static documentation generators and provide a seamless developer experience.

## Key Features

-   **Dynamic Ingestion:** Update your docs via API.
-   **Granular RBAC:** Control access at the document level.
-   **Unified Schema Registry:** Support for OpenAPI, AsyncAPI, and JSON Schema.
-   **Blazing Fast:** Built with Rust, Axum, and Leptos.
